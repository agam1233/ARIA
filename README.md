What aria is
Aria is a Atonomus reasoning intellegent agent, it can write, read, edit, run shell commands, and do more.
What it does -
It speaks to a provider and calls the model
The model is told a system prompt to help it use its tools (if you remove the system prompt you basically lobotomize the model, but because this is opensource and free as in speech and in beer, you can fork it and do whatever you want to the code).
You can add other tools. (You'll need to change the system prompt.)
